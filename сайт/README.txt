Перед тем как продолжить, проверьте, находятся ли исполняемые файлы в одной папке

Чтобы открыть сайт, откройте файл index.html или registr.html

В registr.html содержится регистрационная форма, при регистрации вас перенесёт на index.html файл (главная страница сайта)

Сама же система регистрации и хранения данных пользователя в базе данных ещё не готова, файл registr.html создан по сути для разнообразия

Папка "Исходники" содержит файлы формата JPEG(jpg)

Файл styles.css содержит код для задания цветов, шрифтов, стилей, расположения отдельных блоков и других аспектов представления внешнего вида.
